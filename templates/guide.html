Add item- Add Automatically

Before proceed submiting the excel file, make sure the excel file is '.csv'.
'.xlsx/ .xls' format is unreadable to database.
Save your Excel file as '.csv' and make sure the format of data inside the file is default

Reminder

Save as '.csv'
Data in the file must be in default format
Data in the file cannot be Null/blank, use something such as 'na' to represent Null/blank
The websystem will (skip/not read) header of the file. If one of your data missing, means one of your data is header of the file
